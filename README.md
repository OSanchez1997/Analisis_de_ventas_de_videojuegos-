🎮 Análisis de Ventas de Videojuegos y Tendencias de Mercado

📌 Descripción del proyecto

Este proyecto tiene como objetivo analizar datos históricos de ventas de videojuegos para identificar los factores que determinan el éxito comercial en la industria. A través del análisis de plataformas, géneros, regiones y clasificaciones, se busca generar insights que apoyen la toma de decisiones estratégicas.


🎯 Objetivos

Identificar qué plataformas generan mayores ventas
Analizar los géneros más populares a lo largo del tiempo
Evaluar diferencias de comportamiento por región (NA, EU, JP)
Determinar factores que influyen en el éxito de un videojuego
Apoyar decisiones de negocio basadas en datos

📊 Datos utilizados

El dataset incluye información sobre:

Nombre del videojuego
Plataforma
Año de lanzamiento
Género
Ventas por región (Norteamérica, Europa, Japón, global)
Clasificación (ESRB)

🧠 Metodología

Limpieza y preparación de datos
Tratamiento de valores nulos
Conversión de tipos de datos
Eliminación de duplicados
Análisis Exploratorio de Datos (EDA)
Distribución de ventas por plataforma y género
Análisis de tendencias a lo largo del tiempo
Comparación de ventas entre regiones
Análisis estadístico
Evaluación de diferencias entre grupos
Identificación de patrones relevantes
Visualización de datos
Gráficos para facilitar la interpretación de resultados

🛠️ Herramientas utilizadas

Python
Pandas
NumPy
Matplotlib
Seaborn
SciPy
Jupyter Notebook

📈 Resultados

Se identificaron las plataformas con mayor desempeño en ventas globales
Se detectaron géneros con mayor consistencia en popularidad
Se observaron diferencias claras en preferencias por región
Se identificaron factores clave que influyen en el éxito comercial de los videojuegos


💡 Conclusiones

El análisis permitió entender cómo variables como plataforma, género y región impactan en el rendimiento de un videojuego. Estos hallazgos pueden ser utilizados para orientar decisiones estratégicas en desarrollo, marketing y distribución dentro de la industria.
